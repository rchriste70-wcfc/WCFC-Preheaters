# Preheater Program Calendar  
**Wings of Carolina Flying Club**

This calendar defines the annual and recurring events required to operate,
maintain, and govern the club’s Aircraft Preheater Program.

The calendar is role-based and intended to support continuity across seasons
and volunteer transitions.

---

## Annual Calendar Overview

| Month | Activity | Owner | Notes |
|------|--------|------|------|
| **September** | Program Planning & Inventory Review | Preheater Program Volunteer | Review charter, confirm inventory, assess heater condition |
| **October** | **Season Kickoff** | Preheater Program Volunteer | Fuel refresh, functional test runs, member kickoff communication |
| **November** | In-Season Inspection | Preheater Program Volunteer | Visual inspection, fuel level check |
| **December** | Cold-Weather Readiness Check | Preheater Program Volunteer | Verify availability ahead of peak cold |
| **January** | Mid-Season Inspection & Reminder | Preheater Program Volunteer | Functional test; issue safety reminder |
| **February** | High-Risk Period Check | Preheater Program Volunteer | Increased vigilance during extended cold snaps |
| **March** | Late-Season Inspection | Preheater Program Volunteer | Check wear, fuel condition |
| **April** | Season Shutdown Planning | Preheater Program Volunteer | Notify members of upcoming shutdown |
| **May** | **Season Shutdown & Storage** | Preheater Program Volunteer | Clean, inspect, store per manufacturer guidance |
| **June** | Documentation Review | Preheater Program Volunteer | Update notes, issues, lessons learned |
| **July** | Succession Readiness Check | Board / Volunteer | Confirm documentation completeness |
| **August** | Charter Review & Board Touchpoint | Board / Volunteer | Annual review ahead of next season |

---

## Recurring In-Season Tasks  
*(Typically October through April, weather dependent)*

| Frequency | Task |
|---------|-----|
| Weekly / Bi-Weekly | Fuel level check (maintain ≥ ¼ tank) |
| Monthly | Visual inspection and functional test |
| As Needed | Issue response, out-of-service tagging, member communication |

---

## Event Definitions

### Season Kickoff
- Confirm both preheaters are operational
- Refresh or replace fuel
- Inspect hoses, cords, and accessories
- Publish preseason kickoff communication to members

### Mid-Season Inspection
- Functional test under load
- Verify signage, Quick Start cards, and QR codes
- Address any recurring issues

### Season Shutdown
- Clean preheaters
- Drain or stabilize fuel per manufacturer guidance
- Prepare units for storage
- Communicate shutdown status to membership

---

## Notes
- Preheating is **required** when ambient air temperature is below 40°F.
- Dates may shift slightly based on local weather conditions.
- This calendar should be reviewed annually as part of program governance.
